About Me

My name is Blessing Dawodu, from the moment produced "Hello World" in the console of my first applictation, I knew I was hooked into the world of software engineering.

As a frontend developer I specialize in the development of applications that run in a web browser or on a mobile device. I am responsible for the look and feel of a website or application, and I use HTML, CSS, and JavaScript to build and style the user interface.

I work closely with designers to implement visual designs and with backend developers to integrate with server-side functionality. I also test and debug their code to ensure that it is functional and performs well across different devices and browsers.

When I am not doing anything related to coding, I'm obviously learning new skills on how to stay competitive in the market.
